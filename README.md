# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

### 使い方

1. git clone
   1. `git clone https://github.com/naberina/MannerBooks.git`
2. npm install
   1. `cd MannerBooks && npm install`
3. npx zenn preview
   1. `npx zenn preview`
4. ローカル環境でプレビュー
   1. `localhost:8000`
